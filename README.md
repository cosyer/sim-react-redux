# sim-react-redux
> 简化版的 react-redux ！

## 由于精力有限，目前文档较为简陋，可以在点击 [详细文档地址](https://hannq.github.io/sim-react-redux/) 尝试在线编辑

## 特点&目标

+ 以最简洁的方式，完美支持 typescript。
+ 可以在任意 页面/组件 里面 获取/修改任意 页面 的值。
+ 支持所有的 redux 中间件，充分利用社区资源。
+ api 继承 `react-redux`，上手成本低。
